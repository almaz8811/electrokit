## Что делает

ElectroKit автоматически рассчитывает электрику квартиры, дома или офиса по нормам ПУЭ:

- **Разбивает нагрузки на группы** — освещение, розетки, выделенные линии, кондиционеры
- **Подбирает сечение кабеля** — ВВГнг с учётом типа линии и минимальных сечений по ПУЭ
- **Выбирает номинал автомата** — с коэффициентом запаса 1.25 для мощных нагрузок
- **Определяет мокрые зоны** — автоматически по названию комнаты, ставит дифавтоматы и УЗО
- **Считает падение напряжения** — ΔU% по длине линии с рекомендацией увеличить сечение
- **Проверяет селективность** — предупреждает если групповой автомат ≥ вводному
- **Генерирует схему щитка** — визуальный DIN-рейки с легендой
- **Составляет смету** — материалы и работы с ценами, итого в вашей валюте

## Что рассчитывает

**Группы нагрузки** — для каждой комнаты формируются группы:
- Освещение → 1.5 мм², автомат по нагрузке
- Розетки и приборы → минимум 2.5 мм² + C16
- Стиральная машина, посудомойка, духовка, бойлер → минимум 2.5 мм², ток с коэффициентом 1.25
- Кондиционер → отдельная группа на каждый блок, минимум 2.5 мм²

**Коэффициент спроса (Kd):**
- Квартира — 0.7
- Частный дом — 0.8
- Офис — 0.6

**Мокрые зоны** — определяются по ключевым словам в названии комнаты:
`ван`, `санузел`, `туалет`, `кухн`, `баня`, `душ`, `kitchen`, `bathroom`, `wc`
