## Что изменилось в v3.0

### Трёхфазный режим

Добавлен переключатель типа сети в карточке объекта: однофазная 230В или трёхфазная 400/230В. Пользователь не выбирает фазы вручную — просто добавляет комнаты и приборы как обычно.

После нажатия «Рассчитать» инструмент автоматически распределяет группы по фазам L1/L2/L3 жадным алгоритмом: каждая следующая группа назначается на фазу с наименьшей текущей нагрузкой. Для бытового щита это даёт результат не хуже оптимального.

Блок баланса фаз показывает мощность и ток по каждой фазе с визуальной полосой и оценкой перекоса:
- ✔ Баланс отличный — перекос менее 10%
- ⚠ Перекос умеренный — менее 25%
- ⛔ Перекос высокий — более 25%

В таблице групп появляется колонка «Фаза» с цветовой маркировкой: L1 amber, L2 зелёный, L3 фиолетовый. В щитке вводной автомат показывается как 4P.

### Режим АВТО/МАНУАЛ

Переключатель в таблице результатов после расчёта. В мануальном режиме сечение и автомат для каждой группы редактируются напрямую — смета пересчитывается автоматически. Нарушения ПУЭ подсвечиваются красным, корректные значения — зелёной галочкой. При переключении в АВТО все изменения сбрасываются.

### Духовой шкаф с вилкой

Два варианта в списке приборов: «Духовой шкаф (выделенная линия)» — минимум 4мм², прямое подключение к щитку; и «Духовой шкаф (вилка → розетка)» — идёт в розеточную группу, 2.5мм² + C16. По замечанию комментаторов: духовка с вилкой и встроенная духовка — разные случаи.

### Исправленные баги

- Стиралка и бойлер 2кВт давали C10 по формуле. Добавлен практический минимум C16 для всех выделенных линий.
- УЗО зоны могло получить номинал 10А которого не существует. Добавлен минимум 16А.

## Что изменилось в v2.0

### Исправлена критическая ошибка логики подбора кабеля

В v1 автомат выбирался по расчётному току, а кабель — по автомату. Это приводило к тому, что кабель 2.5мм² мог получить автомат C25, хотя допустимый ток 2.5мм² — 21А. Автомат должен защищать кабель, а не наоборот.

В v2 логика исправлена по ПУЭ:
- **Автомат** выбирается по рабочему току нагрузки (`watts / V`)
- **Кабель** выбирается так, чтобы его допустимый ток был не ниже тока с запасом (`watts × 1.25 / V` для выделенных линий) и не ниже номинала автомата
- Инвариант «автомат ≤ допустимый ток кабеля» соблюдается всегда

Таблица допустимых токов (ПУЭ табл. 1.3.4, медь, одиночная прокладка, +25°C):

| Сечение | Допустимый ток | Макс. автомат |
|---------|---------------|---------------|
| 1.5 мм² | 15 А | B/C 10 |
| 2.5 мм² | 21 А | C 16 |
| 4.0 мм² | 27 А | C 25 |
| 6.0 мм² | 34 А | C 32 |
| 10 мм²  | 50 А | C 50 |

### Исправлена схема защиты мокрых зон

В v1 на мокрую зону одновременно ставилось УЗО зоны и дифавтоматы на каждую линию — логическое противоречие. В v2 две корректные схемы на выбор:

**Схема А (по умолчанию) — дифавтомат на каждую линию.** Срабатывание гасит одну линию. Рекомендуется для большинства объектов.

**Схема Б — УЗО на зону + автоматы.** Одно УЗО на комнату с расчётным номиналом по сумме токов линий за ним. Дешевле от двух линий, но срабатывание гасит всю зону. Переключатель в настройках.

### Противопожарное УЗО убрано из дефолтной схемы

Противопожарное УЗО 100мА — самостоятельное устройство защиты от токов утечки, не связанное с реле напряжения или УЗИП (это принципиально разные устройства). Убрано из дефолта как опциональный элемент — включается в настройках при необходимости.

### Новые типы нагрузки в v2

- Варочная панель — 5 кВт, минимум 4 мм²
- Варочная + духовка (комбо) — 7 кВт, минимум 6 мм²
- Розетка в с/у (фен) — 1500 Вт
- Коэффициент 1.25 применяется только к выбору кабеля, не автомата

## Что делает

ElectroKit автоматически рассчитывает электрику квартиры, дома или офиса по нормам ПУЭ:

- Однофазный и трёхфазный режим — автоматическое распределение групп по фазам L1/L2/L3
- Переключатель 1Ф/3Ф на уровне каждого прибора — для мощных нагрузок с трёхфазным подключением
- Разбивает нагрузки на группы — освещение, розетки, выделенные линии, кондиционеры
- Подбирает сечение кабеля ВВГнг по допустимому току (ПУЭ табл. 1.3.4)
- Выбирает класс и номинал автомата — B10 для освещения, C16+ для остального
- Определяет мокрые зоны автоматически по названию комнаты
- Строит схему защиты мокрых зон — схема А или Б
- Считает падение напряжения ΔU% по длине линии
- Проверяет селективность
- Режим АВТО/МАНУАЛ — ручная корректировка с проверкой ПУЭ и автообновлением сметы
- Генерирует схему щитка (DIN-рейка) с перетаскиванием модулей
- Составляет смету в вашей валюте

## Что рассчитывает

**Группы нагрузки** для каждой комнаты:
- Освещение (точки / люстры) → 1.5мм² + B10
- Розетки и приборы → минимум 2.5мм², автомат по рабочему току
- Розетка в с/у (фен) → 1500 Вт, отдельно от обычных розеток
- Стиральная машина, посудомойка, бойлер → минимум 2.5мм², C16 минимум
- Духовой шкаф (выделенная линия) → минимум 4мм²
- Духовой шкаф (вилка → розетка) → розеточная группа, 2.5мм² + C16
- Варочная панель → минимум 4мм²
- Варочная + духовка → минимум 6мм²
- Тёплый пол (электрический) → выделенная линия, минимум 2.5мм²
- Зарядка электромобиля → выделенная линия, минимум 6мм²
- Кондиционер сплит → выделенная линия, минимум 2.5мм²
- Кондиционер оконный → розеточная группа

**Коэффициент спроса (Kd):**
- Квартира — 0.7
- Частный дом — 0.8
- Офис — 0.6

**Мокрые зоны** по названию комнаты:
- Ванная / баня / душ → 10мА
- Кухня / санузел / туалет → 30мА

**Трёхфазный режим:**
- Жадный алгоритм распределения: каждая группа назначается на фазу с наименьшей текущей нагрузкой
- Показ тока и мощности по каждой фазе
- Оценка перекоса фаз
- Вводной автомат 4P в схеме щитка

**Проверки:**
- Расчётный ток > вводной → ⛔ перегрузка
- Групповой автомат ≥ вводного → ⛔ нарушение селективности
- Разница < 1 ступени → ⚠ низкая селективность
- Розеточная группа > 3500 Вт → 💡 рекомендация разделить

## Предупреждение

> ElectroKit предназначен для **предварительной оценки** нагрузок и ориентировочного подбора оборудования.
> Токи кабелей по ПУЭ табл. 1.3.4 даны для одиночной прокладки в воздухе при +25°C.
> При групповой прокладке, в трубе или при повышенной температуре применяйте понижающие коэффициенты.
> Заземление и тип системы заземления (TN-C-S, TN-S, TT) инструментом не рассчитываются — проверку контура заземления выполняет специалист с измерительным оборудованием.
> Результаты **не являются** проектной документацией.
