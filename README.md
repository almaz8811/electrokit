## Что изменилось в v2.0

### Исправлена критическая ошибка логики подбора кабеля

В v1 автомат выбирался по расчётному току, а кабель — по автомату. Это приводило к тому, что кабель 2.5мм² мог получить автомат C25, хотя допустимый ток 2.5мм² — 21А. Автомат должен защищать кабель, а не наоборот.

В v2 логика исправлена по ПУЭ:
- **Автомат** выбирается по рабочему току нагрузки (`watts / V`)
- **Кабель** выбирается так, чтобы его допустимый ток был не ниже тока с запасом (`watts × 1.25 / V` для выделенных линий) и не ниже номинала автомата
- Инвариант «автомат ≤ допустимый ток кабеля» соблюдается всегда

Таблица допустимых токов (ПУЭ табл. 1.3.4, медь, одиночная прокладка, +25°C):

| Сечение | Допустимый ток | Макс. автомат |
|---------|---------------|---------------|
| 1.5 мм² | 15 А | B/C 10 |
| 2.5 мм² | 21 А | C 16 |
| 4.0 мм² | 27 А | C 25 |
| 6.0 мм² | 34 А | C 32 |
| 10 мм²  | 50 А | C 50 |

### Исправлена схема защиты мокрых зон

В v1 на мокрую зону одновременно ставилось УЗО зоны и дифавтоматы на каждую линию — логическое противоречие. В v2 две корректные схемы на выбор:

**Схема А (по умолчанию) — дифавтомат на каждую линию.** Срабатывание гасит одну линию. Проще монтаж, нагляднее щиток. Рекомендуется для большинства объектов.

**Схема Б — УЗО на зону + автоматы.** Одно УЗО на комнату с расчётным номиналом по сумме токов линий за ним (ближайший стандартный ≥ расчётного). Дешевле от двух линий, но срабатывание гасит всю зону. Переключатель в настройках.

Ванная/баня/душ (10мА) и кухня/санузел (30мА) всегда попадают в разные зоны — объединить их за одним УЗО нельзя из-за разных уставок.

### Противопожарное УЗО 100мА убрано из дефолтной схемы

Противопожарное УЗО без реле напряжения и УЗИП создаёт ложное ощущение «полного» щитка. В v2 это опция «Расширенная защита» в настройках с соответствующим пояснением.

### Новые типы нагрузки

- **Варочная панель** — 5 кВт, минимум 4 мм², отдельная линия
- **Варочная + духовка (комбо)** — 7 кВт, минимум 6 мм², отдельная линия
- **Розетка в с/у (фен)** — 1500 Вт вместо 200 Вт для обычной розетки. Реальная нагрузка фена/плойки в ванной.
- **Духовой шкаф** — минимум 4 мм² независимо от расчётного тока (пусковые токи, нормы новостроек)

### Прочее

- Допустимый ток кабеля по ПУЭ отображается в таблице групп под сечением — пользователь видит на чём основан выбор
- Коридор добавлен в список помещений по умолчанию
- Формула Kd отображается явно в итогах: `18 240 × Kd 0.7 (квартира) = 12 768 Вт`
- Расчёт модулей щитка учитывает выбранную схему и наличие противопожарного УЗО

## Что делает

ElectroKit автоматически рассчитывает электрику квартиры, дома или офиса по нормам ПУЭ:

- Разбивает нагрузки на группы — освещение, розетки, выделенные линии, кондиционеры
- Подбирает сечение кабеля ВВГнг по допустимому току (ПУЭ табл. 1.3.4) с запасом 1.25 для выделенных линий
- Выбирает класс и номинал автомата — B10 для освещения, C16/C25/C32 для остального; автомат никогда не превышает допустимый ток кабеля
- Определяет мокрые зоны автоматически по названию комнаты
- Строит схему защиты мокрых зон — дифавтоматы (схема А) или УЗО на зону + автоматы (схема Б)
- Считает падение напряжения ΔU% по длине линии с рекомендацией по увеличению сечения
- Проверяет селективность — ошибка если групповой автомат ≥ вводного
- Генерирует схему щитка (DIN-рейка)
- Составляет смету — материалы и работы в вашей валюте

## Что рассчитывает

**Группы нагрузки** для каждой комнаты:
- Освещение → 1.5мм² + B10 (класс B для осветительных групп)
- Розетки и приборы → минимум 2.5мм², автомат по рабочему току
- Розетка в с/у (фен) → 1500 Вт, считается отдельно от обычных розеток
- Стиральная машина, посудомойка, бойлер → минимум 2.5мм², кабель с запасом ×1.25
- Духовой шкаф → минимум 4мм², кабель с запасом ×1.25
- Варочная панель → минимум 4мм², кабель с запасом ×1.25
- Варочная + духовка → минимум 6мм², кабель с запасом ×1.25
- Кондиционер → отдельная группа на каждый блок, минимум 2.5мм², кабель с запасом ×1.25

**Коэффициент спроса (Kd)** учитывает что все приборы не работают одновременно:
- Квартира — 0.7
- Частный дом — 0.8
- Офис — 0.6

**Мокрые зоны** определяются по названию комнаты:
- Ванная / баня / душ → 10мА
- Кухня / санузел / туалет → 30мА

**Схема щитка (схема А, по умолчанию):**
1. Вводной автомат
2. Дифавтоматы для мокрых групп (10 или 30мА)
3. Автоматы для сухих групп
4. Опционально: противопожарное УЗО 100мА (требует реле напряжения + УЗИП)

**Схема щитка (схема Б):**
1. Вводной автомат
2. УЗО на каждую мокрую зону (расчётный номинал по сумме токов) + автоматы за ним
3. Автоматы для сухих групп

**Проверки:**
- Расчётный ток > вводной → ⛔ перегрузка
- Групповой автомат ≥ вводного → ⛔ нарушение селективности
- Разница < 1 ступени → ⚠ низкая селективность
- Розеточная группа > 3500 Вт → 💡 рекомендация разделить
- Более 6 розеток на группу → 💡 рекомендация разделить

## Предупреждение

> ElectroKit предназначен для **предварительной оценки** нагрузок и ориентировочного подбора оборудования.
> Токи кабелей по ПУЭ табл. 1.3.4 даны для одиночной прокладки в воздухе при +25°C.
> При групповой прокладке, в трубе или при повышенной температуре применяйте понижающие коэффициенты.
> Результаты **не являются** проектной документацией.
