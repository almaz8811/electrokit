# Браузерный расчётчик электрики: один HTML-файл с подбором кабеля, автоматов, УЗО и сметой — без бэкенда

*Теги: javascript, электротехника, веб-разработка, open source, инструменты*

---

## Предыстория

Когда делаешь ремонт или помогаешь кому-то с электрикой, неизбежно возникает вопрос: какой кабель брать, какой автомат ставить, сколько всего нужно на щиток?

Гуглишь — находишь либо огромные Excel-таблицы непонятного происхождения, либо калькуляторы с одним полем «введите мощность», которые выдают одно число. Ни то, ни другое не помогает составить реальную картину.

Я решил сделать инструмент, который даёт полный ответ: группы нагрузки, кабели, автоматы, УЗО, схему щитка и смету — всё сразу. И чтобы работало без регистрации, без установки, без интернета.

Результат: **ElectroKit** — один HTML-файл, 1200 строк, никаких зависимостей.

**[Попробовать онлайн](https://darkenamber.github.io/electrokit)** | **[GitHub](https://github.com/darkenamber/electrokit)**

## Что умеет

Пользователь добавляет комнаты, в каждой — приборы с типом и количеством. Нажимает «Рассчитать». Инструмент делает следующее:

**Разбивает на группы.** Освещение отдельно, розетки отдельно, каждый мощный прибор на свою линию. Кондиционеры — по одной группе на каждый блок.

**Подбирает кабель.** ВВГнг с учётом типа линии. Розеточные и выделенные группы — минимум 2.5 мм² принудительно, потому что иначе это нарушение ПУЭ. Для мощных нагрузок (стиралка, духовка, бойлер, кондиционер) ток считается с коэффициентом 1.25.

**Подбирает автомат.** По расчётному току. Цветовая индикация: зелёный ≤16А, жёлтый ≤32А, красный >32А.

**Определяет мокрые зоны.** Автоматически по ключевым словам в названии комнаты: «ван», «санузел», «кухн», «баня» и другие. Мокрые группы получают дифавтомат вместо автомата, на каждую мокрую зону добавляется УЗО 25А/30мА.

**Считает падение напряжения.** Если ввести длину линии, сразу появляется ΔU% — зелёный до 4%, жёлтый до 5%, красный с рекомендацией увеличить сечение.

**Проверяет селективность.** Если групповой автомат равен или больше вводного — красная ошибка с объяснением. Если разница меньше одной ступени — жёлтое предупреждение.

**Генерирует схему щитка.** Визуальная DIN-рейка с цветовой легендой: вводной автомат, УЗО вводное, УЗО по зонам, групповые автоматы и дифавтоматы.

**Составляет смету.** Автоматически считает метраж кабеля по сечениям, количество автоматов по номиналам, УЗО, дифавтоматы, розетки, выключатели, подрозетники. Отдельный блок работ: прокладка кабеля, штробление, установка, сборка щитка. Итого в вашей валюте.

**Даёт рекомендации.** Отдельный блок советов: если вводной загружен >80%, если на розеточной группе слишком много точек, если щиток требует >16 модулей.

## Технические решения

### Почему один файл

Это сознательное архитектурное решение, а не лень разбить на модули.

Один файл — это:
- Скинуть коллеге в Telegram одним сообщением
- Открыть на любом устройстве без установки
- Работать на объекте без интернета
- Нет проблем с обновлениями CDN

Единственная «зависимость» — Google Fonts для шрифтов. При отсутствии интернета браузер просто использует системный шрифт.

### Хранение данных

Всё в `localStorage`. Два ключа:

```javascript
localStorage.setItem('ek_proj', JSON.stringify(project)); // проект
localStorage.setItem('ek_cfg',  JSON.stringify(settings)); // цены и настройки
```

Проект сохраняется при каждом изменении. Настройки — при каждом изменении цены. Никаких кнопок «Сохранить».

Дополнительно — импорт/экспорт в JSON. Можно передать проект коллеге или хранить несколько объектов как отдельные файлы.

### Логика подбора кабеля

```javascript
function cableForLoad(watts, lineType) {
  const margin = (lineType === 'dedicated' || lineType === 'ac') ? 1.25 : 1.0;
  const a = (watts * margin) / 220;

  // Минимальное сечение по типу линии
  const minSec = (lineType === 'socket' || lineType === 'dedicated' || lineType === 'ac')
    ? 2.5 : 1.5;

  let section, breaker;
  if (a <= 16)      { section = 1.5; breaker = 16; }
  else if (a <= 25) { section = 2.5; breaker = 25; }
  else if (a <= 32) { section = 4.0; breaker = 32; }
  else if (a <= 40) { section = 6.0; breaker = 40; }
  else              { section = 10;  breaker = 63; }

  if (section < minSec) section = minSec;
  // ...
}
```

Ключевой момент: духовка 2500 Вт даёт расчётный ток 14.2 А (с коэффициентом 1.25) — автомат был бы 16А и кабель 1.5 мм². Но минимум для выделенной линии — 2.5 мм². Без этого ограничения расчёт формально правильный, но на практике неверный.

### Расчёт падения напряжения

```javascript
function voltDrop(watts, lengthM, sectionStr) {
  // ΔU% = (2 × L × I × ρ) / (S × U) × 100
  // ρ = 0.0175 Ω·мм²/м для меди
  const I = watts / 220;
  return (2 * lengthM * I * 0.0175) / (parseFloat(sectionStr) * 220) * 100;
}
```

Если ΔU > 5% — красный индикатор и подсказка с увеличенным сечением.

### Автоматическое определение мокрых зон

```javascript
const WET_KW = ['ван', 'санузел', 'туалет', 'кухн', 'баня', 'душ',
                'bathroom', 'kitchen', 'toilet', 'wc'];

function isWetRoom(name) {
  const n = name.toLowerCase();
  return WET_KW.some(k => n.includes(k));
}
```

Пользователь называет комнату «Ванная» или «Санузел» — инструмент автоматически ставит дифавтомат и УЗО. Никакого ручного флага.

## Что не считает

Честно о границах инструмента:

- Трёхфазная сеть (планируется в v2)
- Реактивная мощность и cos φ
- Термическая стойкость кабеля при КЗ
- Уставки по времени срабатывания автоматов
- Расчёт токов КЗ

ElectroKit — инструмент для **предварительной оценки**, не замена проектной документации.
