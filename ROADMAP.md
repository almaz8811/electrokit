# Roadmap — ElectroKit

Небольшие улучшения и фиксы по комментариям сообщества:

- [ ] Исправить дисклеймер про противопожарное УЗО
- [ ] Улучшить UX трёхфазного режима по результатам тестирования
- [ ] Дополнительные типы нагрузки по запросам пользователей

---

## v4.0 — визуализация щитка

Главная фича v4 — интерактивная схема щитка.

**Что планируется:**
- Перетаскивание модулей на DIN-рейке
- Ручная коммутация — назначить любой автомат на любую группу
- Визуальный подбор размера щитка по количеству модулей
- Однолинейная схема в SVG — экспорт для документации
- Группировка по DIN-рейкам для многосекционных щитков

---

## Не планируется

- Расчёт заземления и токов КЗ — требует измерений на объекте
- Алюминиевый кабель — не актуально для жилых объектов
- Трёхфазные UPS и генераторы — отдельный инструмент UPSPowerKit
- Конкретные модели оборудования — инструмент независимый от вендоров
