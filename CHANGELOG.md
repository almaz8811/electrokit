# Changelog — ElectroKit

## v3.0

### Новое
- Трёхфазный режим 400/230В — переключатель в карточке объекта
- Автоматическое распределение групп по фазам L1/L2/L3 (жадный алгоритм)
- Блок баланса фаз — мощность и ток по каждой фазе, оценка перекоса
- Рекомендация по улучшению баланса — конкретный совет что перенести
- Вводной автомат 4P в схеме щитка для трёхфазной сети
- Переключатель 1Ф/3Ф на уровне каждого прибора (выделенные линии и кондиционеры)
- Режим АВТО/МАНУАЛ — ручная корректировка сечения и автомата с проверкой ПУЭ
- Смета пересчитывается автоматически при изменении в мануальном режиме
- Редактируемая мощность для каждого прибора — типовое значение по умолчанию, можно изменить
- Предупреждение при вводе мощности >2000Вт в розеточную группу
- Новые типы нагрузки: тёплый пол, зарядка электромобиля, кондиционер сплит и оконный
- Два варианта духового шкафа: выделенная линия (мин. 4мм²) и с вилкой в розетку
- Кнопка «Очистить» в шапке с подтверждением
- Переключатель схемы мокрых зон перенесён в правую колонку настроек
- Баннер «нажмите Рассчитать» при изменении настроек когда результаты уже показаны
- GitHub / Patreon / Ko-fi в шапке

### Исправлено
- Минимум C16 для выделенных линий (стиралка, бойлер давали C10)
- Минимум 16А для номинала УЗО зоны
- Баг с id приборов после загрузки проекта — смена типа меняла не тот прибор
- Сортировка нагрузок по убыванию мощности перед распределением по фазам

## v2.0

### Исправлено (критическое)
- Логика подбора кабеля — переделана полностью. Было: автомат по расчётному току → кабель по автомату. Стало: автомат по рабочему току → кабель по допустимому току с запасом. Кабель 2.5мм² больше не получает C25
- Схема защиты мокрых зон — убрано противоречие УЗО зоны + дифавтоматы за ним
- Коэффициент 1.25 применяется только к выбору кабеля, не к автомату

### Новое
- Две схемы защиты мокрых зон на выбор: схема А (АВДТ на линию, по умолчанию) и схема Б (УЗО на зону + автоматы)
- Номинал УЗО зоны рассчитывается по сумме токов линий за ним
- Противопожарное УЗО 100мА убрано из дефолта — опция «Расширенная защита» в настройках
- Минимальные сечения по типу прибора: духовка 4мм², варочная 4мм², комбо 6мм²
- Новые типы нагрузки: варочная панель, варочная+духовка, розетка в с/у (фен)
- Допустимый ток кабеля отображается в таблице групп
- Явный показ формулы Kd в итогах
- Коридор в помещениях по умолчанию
- Заземление добавлено в дисклеймер
